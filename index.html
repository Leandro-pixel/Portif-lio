JavaScript é uma linguagem de programação de alto nível, leve e interpretada que foi criada para adicionar interatividade a páginas da web. Ela é uma das principais tecnologias usadas no desenvolvimento front-end e também é amplamente utilizada no desenvolvimento de aplicativos do lado do cliente e do lado do servidor.

HTML (HyperText Markup Language) é a linguagem padrão usada para estruturar e apresentar conteúdo na web. É uma linguagem de marcação que define a estrutura de uma página da web, especificando elementos e sua organização hierárquica.

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o design de páginas da web escritas em HTML (ou outra linguagem de marcação). Ele permite controlar a aparência, o layout e o estilo de elementos individuais ou grupos de elementos em uma página.

O Node.js é um ambiente de execução JavaScript do lado do servidor, construído com base no mecanismo V8 do Google Chrome. Ele permite que você execute código JavaScript fora do navegador, permitindo a criação de aplicativos web escaláveis e de alto desempenho.

O React é uma biblioteca JavaScript de código aberto para a construção de interfaces de usuário (UI) interativas e reativas. Desenvolvido pelo Facebook, o React permite criar componentes reutilizáveis e declarativos, que são eficientes e fáceis de manter.

CountDown/Formulário

É um site simulando a estréia de um lançamento de um produto fictício, faz parte de uma prova didática criada pela Rocketseat. Incrementei ao desáfio um formulário com validação para que o cliente receba notificações em seu e-mail sobre o lançamento.

Landing Page

Site estilo landing page sobre NTF's, também prova didática fornecida pela Rocketseat, básicamente foi usado somente HTML e CSS, porém para o efeito de scroll gerador de conteúdo foi utilizado Javascript.

OpenWeather API

Aplicativo utilizando API da Openweather com objetivo de informação climática de cidades/estados ou países. Para a comunicação entre o aplicativo e a API foi utilizado o método Fetch e Json.
